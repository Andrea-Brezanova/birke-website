# Birke Garden Care Website

This is the one-page Next.js website for Birke Garden Care.

## Contact Form Email Setup (Resend)

1. Install Resend:

```bash
npm install resend
```

2. Create `.env.local` in the project root.
3. Add your API key:

```bash
RESEND_API_KEY=your_key_here
```

4. Restart the dev server:

```bash
npm run dev
```

5. Before deploying to Vercel, add `RESEND_API_KEY` in your Vercel project settings.

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
